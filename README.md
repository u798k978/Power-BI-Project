## Project Overview
This project analyzes the sales performance of Wichita Cookie Company using Power BI. The goal is to transform raw sales, cost, product, and country data into an interactive 
dashboard that provides insights into sales trends, profitability, and geographic performance.

## Tools & Technologies
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Visualization & Business Analytics

## Dashboard Features
- Year-to-Date (YTD) Sales and Profit
- Monthly Sales Growth Analysis
- Product-wise Profitability
- Geographic Profit Distribution (Map Visualization)
- Interactive tables for detailed data exploration
- Population vs Sales comparison by country
